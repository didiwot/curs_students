# curs_students

main.cpp:
1. Создаем вектор студентов(Студенты наш класс определенный в ./Student/Student/Student.h)
2. Создание объекта jsonLoader для работы с файлом JSON, который будет использоваться для базы данных студентов.
3. Запись данных о студентах в файл JSON с помощью метода writeStudentDataToJsonFile объекта jsonLoader.
4. Чтение данных о студентах из файла JSON с помощью метода readStudentDataFromJsonFile объекта jsonLoader.
5. Создание динамического объекта interfaceStudents класса Student.
6. Создание динамического объекта ICrypto класса Crypto.
7. Вывод меню пользовательского интерфейса и ввод выбора.
8. В зависимости от выбора пользователя:
- Редактирование данных студента (ввод индекса, редактирование, запись в файл).
- Добавление нового студента (добавление, запись в файл).
- Удаление студента (ввод индекса, удаление, запись в файл).
- Печать данных о студентах.
- Шифрование текста данных о студентах и вызов функции шифрования.
- Дешифрование текста данных и вызов функции дешифрования.
- Выход из программы при выборе 7.
9. Освобождение памяти и завершение программы.

Student.h:
Класс Student представляет студента и содержит следующие атрибуты:

fullName - полное имя студента;
dateOfBirth - дата рождения студента;
enrollmentYear - год поступления студента;
faculty - факультет студента;
department - кафедра студента;
group - группа студента;
studentID - уникальный идентификатор студента;
gender - пол студента;
grades - оценки студента в разных сессиях.
Класс Student также содержит следующие методы:

addInfoOneStudent() - добавить информацию о студенте;
addInfo(int index, std::vector<Student> students) - добавить информацию о студенте по индексу;
editStudents(int index, std::vector<Student> &students) - изменить студента по индексу;
addStudent(std::vector<Student>& students) - добавить студента;
deleteStudent(int index, std::vector<Student> &students) - удалить студента по индексу;
showMenu() - показать меню;
printStudents(std::vector<Student> students) - распечатать список студентов;
addGrades(std::vector<std::pair<std::string, int>> sessionGrades) - добавить оценки в текущую сессию;
dataOutTxt(std::vector<Student> students) - экспортировать данные о студентах в текстовый файл;
dataInTxt() - импортировать данные о студентах из текстового файла.
Класс Student также использует макрос NLOHMANN_DEFINE_TYPE_INTRUSIVE из библиотеки nlohmann/json для работы с JSON-данными.

Crypto.h:
Класс Crypto представляет собой класс для шифрования и дешифрования текста с использованием криптографических алгоритмов в Windows.

В классе Crypto есть следующие атрибуты:

dwIndex - индекс для криптографического провайдера;
dwType - тип криптографического провайдера;
cdName - имя криптографического провайдера;
pszName - указатель на имя криптографического провайдера;
x - указатель на дополнительную информацию;
hProv - дескриптор криптографического провайдера;
hSessionKey - дескриптор сессионного ключа шифрования.
Класс Crypto также содержит следующие методы:

Encrypt(string text) - метод для шифрования текста;
Decrypt() - метод для дешифрования текста.
Конструктор Crypto() и деструктор ~Crypto() используются для инициализации и освобождения ресурсов класса.

jsonLoader.cpp:
Класс jsonLoader представляет собой класс, который отвечает за чтение и запись данных о студентах в формате JSON.

В классе jsonLoader есть следующие атрибуты и методы:

jsonFile - путь к JSON-файлу, в котором хранятся данные о студентах;
Конструктор jsonLoader() - конструктор по умолчанию;
writeStudentDataToJsonFile(const std::vector<Student>& students) - метод для записи данных о студентах в формате JSON. Он принимает вектор студентов и сохраняет их в JSON-файле;
readStudentDataFromJsonFile() - метод для чтения данных о студентах из JSON-файла. Он возвращает вектор студентов, прочитанных из файла.
Для работы с JSON используется библиотека nlohmann/json. Класс также имеет приватное наследование от класса Student, что означает, что jsonLoader может использовать приватные члены класса Student.
